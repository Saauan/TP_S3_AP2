list1 module¶

list module

author:FIL - Faculté des Sciences et Technologies - Univ. Lille <http://portail.fil.univ-lille1.fr>_
date:2017, september

Provides

  • a class List for non mutable lists
  • an exception ListError

Lists are either empty, either essentially objects with two composants :

  1. a head composant which represent the head value of the list,
  2. and a tail composant which is represent the tail of the list

Parameters:args (tuple) –

Raise:ListError if UC is not satisfied
